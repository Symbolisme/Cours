Définitions

Le code HTML est consitué de blocs de code encadré par des balises, il existe deux types de balises :

  1. les balises encadrant du code, la balise fermente commence par < /balise >
  2. les balises orphelines sont des balises qui n'encadrent aucun code et elles se ferment elle-mêmes.
    Par exemple <br/> ; <hr/>

Le code HTML permet de créer des blocs de code avec des fonctions particulières
