Définitions

Le code HTML permet de créer des blocs de code encadrés par des balises, il existe deux types de balises :

  1. Les balises encadrantes du code, la balise fermante commence par </balise>.
  2. Les balises orphelines sont des balises qui n'encadrent aucun code et elles se ferment elles-même.
    Par exemple : <br/> , <hr/>

Le code HTML permet de créer des blocs de code avec des fonctions particulières.
